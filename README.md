# Supply Chain Analytics Project

This repository contains the files and documentation for a **Supply Chain Analytics Report**. The project provides a comprehensive analysis of supply chain operations, focusing on key performance indicators (KPIs) to drive strategic recommendations.


## Project Overview

This analytics project was prepared for **Smart Retail Pvt. Ltd.**  and provides data-driven insights across various aspects of the supply chain, including inventory, orders, transportation, and supplier performance.

**Analyst:** Nidhi Jethva 
**Analysis Date:** December 2025 
**Dataset Scope:** The analysis covers data involving **104 Products** and **12 Suppliers** across different **Regions**.

The initial overview of the supply chain reveals the following key metrics:

**Total Sales:** $\$32.69$
**Average Lead Time:** $\$7.33$
**On-time Delivery %:** $29.68\%$
**Total Products Analyzed:** $\$104.00$


## Areas of Analysis

The report is structured to cover the following critical areas of the supply chain:

1. **Supply Chain Overview Dashboard:** Summarizing total orders, sales, average lead time, on-time delivery %, and top products.
2. **Inventory & Warehouse Analysis**
3. **Product Details Analysis**
4. **Transportation & Freight Cost Analysis:** Analyzing shipment volume, delayed orders, **Total Freight Cost** ($\$3.11$), freight cost trends, and regional distribution.
5. **Supplier Performance Analysis:** Evaluating metrics such as **Supplier On-Time %** ($29.83\%$) , lead times, supplier scores, and order distribution.
6. **Cross-Functional Insights**
7. **Key Findings**
8. **Strategic Recommendations**

## How to View the Report

The interactive dashboard requires **Microsoft Power BI Desktop** to be installed:

1.  Clone this repository.
2.  Download and install [Power BI Desktop](https://powerbi.microsoft.com/en-us/desktop/).
3.  Open the `SUPPLY_CHAIN_ANLYTICS.pbix` file using Power BI Desktop.
4.  Navigate through the various dashboard pages (e.g., Overview, Inventory, Suppliers, Transportation) to explore the data visualizations.
